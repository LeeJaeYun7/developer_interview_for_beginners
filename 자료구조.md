### 206. Reversed Linked List?

+ ```
  class Solution {
    public ListNode reverseList(ListNode head) {
        
        ListNode res = null;

        while(head != null){
            ListNode tmp = new ListNode(head.val);
            tmp.next = res;
            res = tmp;
            head = head.next;
        }

        return res; 
    }
 }
   
  ```
