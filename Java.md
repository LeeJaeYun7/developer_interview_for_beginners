<details>
   <summary> 답안 보기 (👈 Click)</summary>
<br />

-----------------------
+ 구체적인 클래스 타입을 모르더라도, 해당 클래스의 정보에 접근할 수 있게 해주는 자바 API입니다.

  리플렉션 API는 클래스명만 알면 Method Area에서 해당 클래스에 대한 정보를 가져옵니다.
